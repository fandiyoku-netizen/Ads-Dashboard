# 📊 Mobile App Campaign Dashboard

An interactive dashboard to analyze and simulate mobile advertising campaign performance.

## 🚀 Features

* KPI Analysis (CPI, ROAS)
* Campaign comparison
* Budget simulation
* Interactive chart visualization

## 🧠 Insights

* Campaign C delivers highest ROAS (4.2)
* Campaign B has highest CPI (inefficient)
* Budget optimization improves install volume

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Chart.js
* Python (data analysis)
